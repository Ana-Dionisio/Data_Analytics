# 📚 Data Analytics Portfolio — SQL, MongoDB, Power BI, Python & Project  
Comprehensive repository of end‑to‑end data analytics projects

This repository contains all major modules developed throughout the Data Analytics program, covering relational databases, NoSQL systems, business intelligence dashboards, Python-based data processing, and an applied end‑to‑end project.  
Each module is organized into its own directory with clear documentation, consistent naming, and reproducible workflows.

The repository reflects real‑world analytics pipelines:  
**data ingestion → cleaning → modeling → analysis → visualization → reporting.**
