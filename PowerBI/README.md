# 📊 Power BI Module — Data Modeling, DAX & Interactive Dashboards  
Comprehensive documentation of Power BI work across Sprints 7 and 8

This directory contains all work developed for the **Power BI module**, covering data preparation, modeling, DAX calculations, KPI creation, and the design of interactive dashboards.  
The goal of this module is to demonstrate the ability to transform raw datasets into clear, actionable business insights through visual analytics.

The module is divided into two stages:

- **Sprint 7** — Data modeling, cleaning, and foundational visuals  
- **Sprint 8** — Advanced dashboards, KPIs, navigation, and interactivity

## 🧭 Overview of the Power BI Module

### **📌 Sprint 7 — Data Preparation & Base Visualizations**

Focus areas:

- Importing and cleaning datasets using Power Query  
- Creating relationships and building a **star schema**  
- Designing the initial data model  
- Creating foundational visuals (tables, bar charts, line charts)  
- Implementing basic DAX measures:
  - `SUM()`
  - `AVERAGE()`
  - `COUNT()`
- Establishing the layout and structure for the final dashboard  

This sprint builds the analytical foundation for the advanced dashboards developed in Sprint 8.

---

### **📌 Sprint 8 — Advanced Dashboards & KPIs**

Focus areas:

- Creating advanced DAX measures:
  - `CALCULATE()`
  - `FILTER()`
  - `TOTALYTD()`
  - YoY and MoM comparisons  
- Designing KPI cards and performance indicators  
- Implementing slicers, filters, drill‑down and drill‑through interactions  
- Creating multi‑page dashboards  
- Adding bookmarks and navigation buttons  
- Applying consistent formatting, themes, and visual storytelling techniques  

This sprint simulates real‑world BI development, focusing on clarity, usability, and business value.
